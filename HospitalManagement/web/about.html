Hospital

Dashboard Pacienti Medici Programari Internari Retete Facturi Raport

Despre aplicatie

Proiect universitar POO pentru gestiunea unui spital.

  • Scop: aplicatia gestioneaza pacienti, medici, programari, internari, facturi si retete.
  • Tehnologii: C++, HTML, CSS, JavaScript, Makefile si Git.
  • Arhitectura: logica principala este implementata in C++, iar HTML/CSS/JavaScript este folosit doar pentru interfata vizuala.
  • Concepte POO: clase si obiecte, incapsulare, mostenire, polimorfism, exceptii personalizate si STL.
  • Rulare: aplicatia C++ genereaza fisiere JSON/CSV, iar interfata web le afiseaza pentru prezentare.
